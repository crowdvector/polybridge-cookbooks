# Asset allocation from prediction market implied probabilities

Use PolyBridge Forecast outputs as probability-weighted inputs to a portfolio optimizer.

## What this cookbook will build

A workflow that:

- fetches macro scenario probabilities
- asks conditional asset-return questions
- computes a covariance matrix from price history
- produces an example optimized allocation

## Files

- `portfolio-allocation.ipynb` - Colab notebook, coming next.
- `setup.sh` - local setup.
- `PROMPT.md` - agent prompt for adapting the cookbook.
- `assets/` - generated charts and snapshot output.
